# 国家职业资格目录

## 2021年版《国家职业资格目录》

2021年版国家职业资格目录共计72项职业资格。
其中，专业技术人员职业资格59项，含准入类33项，水平评价类26项；技能人员职业资格13项。

目录中准入类职业资格关系公共利益或涉及国家安全、公共安全、人身健康、生命财产安全，均有法律法规或国务院决定作为依据；
水平评价类职业资格具有较强的专业性和社会通用性，技术技能要求较高，行业管理和人才队伍建设确实需要。
